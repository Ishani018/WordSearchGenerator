# Font Files

Place the following TTF font files in this directory:

## Required Font Files:

### Roboto
- `Roboto-Regular.ttf` - Download from: https://fonts.google.com/specimen/Roboto
- `Roboto-Bold.ttf` - Download from: https://fonts.google.com/specimen/Roboto

### Open Sans
- `OpenSans-Regular.ttf` - Download from: https://fonts.google.com/specimen/Open+Sans
- `OpenSans-Bold.ttf` - Download from: https://fonts.google.com/specimen/Open+Sans

### Lora
- `Lora-Regular.ttf` - Download from: https://fonts.google.com/specimen/Lora
- `Lora-Bold.ttf` - Download from: https://fonts.google.com/specimen/Lora

### Playfair Display
- `PlayfairDisplay-Regular.ttf` - Download from: https://fonts.google.com/specimen/Playfair+Display
- `PlayfairDisplay-Bold.ttf` - Download from: https://fonts.google.com/specimen/Playfair+Display

## How to Download:

1. Visit each Google Fonts page above
2. Click "Download family" or "Download all styles"
3. Extract the ZIP file
4. Copy the `.ttf` files to this directory with the exact names listed above
